# MyAgent — Agent Coding Desktop

基于大语言模型 API 的桌面端 Agent Coding 工具。

## 技术栈

- **桌面容器**：Electron
- **前端**：React + TypeScript
- **构建工具**：electron-vite + Vite
- **测试**：Vitest

## 当前阶段

阶段 0 · 项目初始化与基础工程

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发模式
npm run dev

# 类型检查
npm run typecheck

# 运行测试
npm run test

# 构建
npm run build
```
