# CodeZ - Agent Coding Desktop

基于大语言模型 API 的桌面端 Agent Coding 工具。

## 技术栈

- **桌面容器**：Tauri 2
- **前端**：React + TypeScript
- **后端**：Rust
- **构建工具**：Tauri + Vite
- **测试**：Vitest

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发模式
npm run dev

# 类型检查
npm run typecheck

# 运行测试
npm run test

# 构建
npm run build
```
